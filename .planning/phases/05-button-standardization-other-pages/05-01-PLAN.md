  <action>
    **HTML changes (settings-page.component.html):**

    Replace the div-based Restart button in #commands section:

    FROM:
    ```html
    <div class="button" appClickStopPropagation [attr.disabled]="commandsEnabled ? null : true"
         (click)="!commandsEnabled || onCommandRestart()">
        <img src="assets/icons/refresh.svg" />
        <span class="text">Restart</span>
    </div>
    ```

    TO:
    ```html
    <button class="btn btn-primary" type="button"
            [disabled]="!commandsEnabled"
            (click)="onCommandRestart()">
        <img src="assets/icons/refresh.svg" alt="" />
        <span class="text">Restart</span>
    </button>
    ```

    Key changes:
    - Change div to button element
    - Add class="btn btn-primary" (positive action = primary)
    - Add type="button"
    - Remove appClickStopPropagation (not needed on button)
    - Change [attr.disabled]="...? null : true" to [disabled]="!commandsEnabled"
    - Remove click guard (!commandsEnabled ||)
    - Add alt="" to img for accessibility

    **SCSS changes (settings-page.component.scss):**

    Update #commands .button selector to #commands .btn:

    FROM:
    ```scss
    .button {
        @extend %button;

        flex-direction: row;
        padding: 10px;
        margin-right: 8px;
        height: 40px;
        font-size: 120%;

        img {
            width: 20px;
            height: 20px;
            filter: invert(1.0);
        }

        .text {
            margin-left: 5px;
        }
    }
    ```

    TO:
    ```scss
    .btn {
        display: flex;
        flex-direction: row;
        align-items: center;
        padding: 10px;
        margin-right: 8px;
        height: 40px;
        font-size: 120%;

        img {
            width: 20px;
            height: 20px;
            filter: invert(1);
        }

        .text {
            margin-left: 5px;
        }
    }
    ```

    Key changes:
    - Change selector from .button to .btn
    - Remove @extend %button
    - Add display: flex and align-items: center (Bootstrap buttons are inline-flex by default)
    - Keep all other custom layout properties
  </action>
