SeedSync
v{{version}}
A tool to sync files from a remote Linux server (like your seedbox) using LFTP for fast transfers.
Features
  • Built on LFTP, the fastest file transfer program
  • Web UI - track and control transfers from anywhere
  • Automatically extract files after sync
  • Auto-Queue based on pattern matching
  • Delete local and remote files easily
Supported Platforms
  • Linux (amd64 and arm64)
  • Raspberry Pi (v2, v3, v4, v5)
  • Windows (via Docker)
  • macOS (via Docker)
